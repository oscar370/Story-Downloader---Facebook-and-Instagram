## Development

### Requirements

- Node.js & pnpm

### Steps

Clone repository:

```bash
git clone https://github.com/oscar370/story-downloader-facebook-and-instagram.git
```

Install dependencies

```bash
pnpm i
```

Start the script in development mode

```bash
pnpm watch
```

The script will be created inside `dist`, copy it to Tampermonkey.
